# Secret Information Storing System

This is a Python project for a Secret Information Storing System using the tkinter library for the graphical user interface. This system allows you to encode and decode secret information within images.

## Features

- Encode: Embeds text messages within an image.
- Decode: Extracts hidden text messages from an encoded image.

## How to Use

1. Run the Python script.
2. Choose either the "Encode" or "Decode" option.
3. Follow the instructions on the respective screen.

### Encode

1. Select an image in which you want to hide text.
2. Enter the text you want to hide.
3. Click the "Encode" button.
4. Save the encoded image.

### Decode

1. Select an image with hidden text.
2. Click the "Select" button.
3. The hidden text will be displayed on the screen.

## Prerequisites

- Python 3.x
- tkinter
- PIL (Python Imaging Library)

## Usage

1. Clone or download the repository to your local machine.
2. Make sure you have Python and the required libraries installed.
3. Run the Python script (`secret_information_system.py`) in your terminal or code editor.
